  KNN Classification 

  Objective
To implement K-Nearest Neighbors (KNN) algorithm for classification.

   Dataset
Iris Dataset

   Steps
- Loaded dataset manually using pandas
- Normalized data using StandardScaler
- Applied KNN algorithm
- Tested different values of K
- Evaluated using accuracy score and confusion matrix

   Output
Achieved high accuracy using KNN model.

   Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
